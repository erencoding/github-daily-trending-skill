# GitHub Daily Trending Skill

📈 OpenClaw skill for fetching GitHub daily trending repositories.

## Overview

Scrapes [GitHub Trending](https://github.com/trending) page using Playwright and generates formatted report with:
- Daily top starred repositories
- Language tags
- Star counts and descriptions

## Usage

When user mentions: `GitHub每日热门`, `每日趋势`, `GitHub trending`, `日热门仓库`

## Features

- Playwright-based web scraping
- Formatted markdown tables
- Daily trending data

## Requirements

- OpenClaw agent
- Playwright installed

## Installation

This skill is managed by OpenClaw. Install via:
```bash
openclaw skills install github-daily-trending
```
